# Everything Is An Expression
## If Expression
```
val a: Int = 10
val b: Int = 20
val x:  = if (a > b) -> Int {
  return a
} else {
  return b
}
```
## While Expression
```
val x: Int = 0
val y: [Int] = while (x < 10) -> Int {
  return x
}
```
## For Expression
```
val a: [Int] = [10, 50, 20, 10]
val y: [Int] = for (x: Int in a) -> Int {
  return x + 10
}
```
