# grime

Repository fow the grime compiler and standard libraries.

## building

### dependencies

#### Ubuntu

`sudo apt-get install cmake llvm git gcc`

#### Arch

`sudo pacman -S cmake llvm git gcc`

### setup

```
git clone https://github.com/Edward-0/grime.git
cd grime
mkdir build
```

### make

```
cd build
cmake ..
make
```

### install

`make install`
